## 简介
数据库系统原理大作业：图书管理系统`BMS`

## 预期使用的技术栈
- 前端：`Electron`+`Create-react-app`+`Material-UI`
- 后端：`Express`+`MySQL Server`

## Building
运行程序
```
$ yarn run build
$ yarn run dev
```
在浏览器中进行热调试
```
$ yarn start
```
