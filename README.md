数据库系统原理大作业：图书管理系统`bms`

- 前端：`Electron`+`Create-react-app`+`Material-UI`

## Building
运行程序
```
$ npm run build
$ npm run electron-start
```
在浏览器中进行热调试
```
$ npm start
```
